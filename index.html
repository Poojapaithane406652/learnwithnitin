C Programming Language.

It was created in the 1970s by Dennis Ritchie, and remains very widely used and influential. By design, C's features cleanly reflect the capabilities of the targeted CPUs.

C++ Programming Language

Created by Danish computer scientist Bjarne Stroustrup as an extension of the C programming language, or "C with Classes".

JavaScript Programming Language.

That is one of the core technologies of the World Wide Web, alongside HTML and CSS. As of 2022, 98% of websites use JavaScript on the client side.

JQuery JS Library

jQuery is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling.

HTML5 -Hypertext Markup Language.

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

CSS3 -Cascading Style Sheets
